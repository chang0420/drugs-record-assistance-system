<h1>居家用藥輔助系統</h1>
協助長者記錄各科就診所開立的藥品資訊，提醒按時服藥，並紀錄用藥反應，也可根據生理量測記錄做紀錄，同時協助醫師整合並調整處方藥物。
<h2>App系統功能</h2>
<ul>
  <li>記錄個人用藥</li>
  <li>用藥提醒</li>
  <li>用藥異常反應紀錄</li>
  <li>常用藥局搜尋</li>
  <li>藥品查詢</li>
  <li>生理數據查看</li>
</ul>
<h2>Web系統功能</h2>
<ul>
  <li>查看生理數據</li>
  <li>用藥異常反應紀錄</li>
  <li>查看個人藥單</li>
</ul>
